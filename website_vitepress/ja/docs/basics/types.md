# プリミティブ型

## 整数型

```v
// 符号付き整数
i8    // -128 to 127
i16   // -32,768 to 32,767
int   // -2,147,483,648 to 2,147,483,647  （常に32ビット）
i64   // -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807

// 符号なし整数
u8    // 0 to 255  （バイトとしても使用）
u16   // 0 to 65,535
u32   // 0 to 4,294,967,295
u64   // 0 to 18,446,744,073,709,551,615

// プラットフォーム依存
isize // 符号付き、ポインタのサイズ
usize // 符号なし、ポインタのサイズ
```

> **注意：** CやGoとは異なり、Vの`int`はプラットフォームに関わらず常に32ビットです。

## 浮動小数点型

```v
f32   // 単精度（32ビット）
f64   // 倍精度（64ビット）
```

## 論理型

```v
is_ready := true
is_done  := false
```

## ルーン

`rune`はUnicodeコードポイントを表します：

```v
letter := `A`        // ルーンリテラルにはバッククォートを使用
emoji  := `🌎`
println(letter)      // A
println(int(letter)) // 65
```

## 型の昇格

小さな型は、同じ演算子の同じ側でより大きな型と組み合わせると自動的に昇格されます：

```v
u := u16(12)
v := 13 + u    // v は u16 — 昇格なし
x := f32(45.6)
y := x + 3.14  // y は f32 — 昇格なし
a := 75        // int（整数リテラルのデフォルト）
b := 14.7      // f64（浮動小数点リテラルのデフォルト）
```

## 数値リテラル

```v
n1 := 1_000_000      // 読みやすさのためのアンダースコア
n2 := 0xff           // 16進数
n3 := 0o77           // 8進数
n4 := 0b1111_0000    // 2進数
f1 := 3.14_159_265   // アンダースコア付き浮動小数点
```

## `voidptr`

主にC相互運用性のために使用される生のポインタ型です。純粋なVコードでは避けてください。

```v
// 主にC相互運用
p := voidptr(0)
```
